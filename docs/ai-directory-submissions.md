## 提交后怎么确认外链生效？

每条提交后，等 2-7 天审核。通过后：

1. 打开对应目录的 SnapProHead 页面
2. 右键 → 查看网页源代码
3. 搜索 `snapprohead.com`
4. 找到链接后看 `<a>` 标签有没有 `rel="nofollow"`：
   - 没有 → dofollow 外链 ✅
   - 有 → nofollow 外链（也有流量价值，SEO 权重稍低）
